# data-collection
Proyecto de análisis de datos para Zuber enfocado en identificar patrones de movilidad y preferencias de los usuarios en Chicago. 

El proyecto incluyó extracción y análisis de datos climáticos y de viajes en taxi mediante SQL y Python, explorando el comportamiento de las compañías de taxis, los barrios con mayor demanda y la relación entre las condiciones climáticas y la duración de los viajes. Además, se realizaron visualizaciones, análisis exploratorio y pruebas de hipótesis para evaluar el impacto de los sábados lluviosos en los tiempos de viaje hacia el Aeropuerto O’Hare.

Paso 1. Extracción de datos

Se recopilaron y analizaron datos climáticos de Chicago y datos de viajes en taxi utilizando consultas SQL y fuentes web.

Paso 2. Análisis exploratorio con SQL

Se analizaron los viajes realizados por diferentes compañías de taxis, identificando las empresas con mayor número de trayectos y comparando el comportamiento de ciertas compañías durante periodos específicos.

Paso 3. Preparación y análisis de hipótesis

Se combinaron datos de viajes y condiciones climáticas para estudiar cómo el clima afecta la duración de los trayectos entre el Loop y el Aeropuerto O’Hare, especialmente durante sábados lluviosos.

Paso 4. Análisis exploratorio con Python

Se importaron y limpiaron datasets en Python, verificando tipos de datos y realizando visualizaciones sobre compañías de taxis y barrios con mayor número de finalizaciones de viajes, obteniendo insights sobre la demanda de movilidad.

Paso 5. Prueba de hipótesis

Se aplicaron pruebas estadísticas para determinar si la duración promedio de los viajes cambia significativamente en condiciones de lluvia durante los sábados.
